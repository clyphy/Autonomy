### Step 4: Create Simple Memory Tracker
**File: `student_memory.py`**
```python
import json
import os
from datetime import datetime
class StudentMemory:
def __init__(self, student_name="student"):
self.student_name = student_name
self.memory_file = f"memory_{student_name}.json"
self.load_memory()
def load_memory(self):
"""Load existing memory or create new"""
if os.path.exists(self.memory_file):
with open(self.memory_file, 'r') as f:
self.memory = json.load(f)
else:
self.memory = {
"student_name": self.student_name,
"sessions": [],
"topics_learned": [],
"struggles": [],
"wins": []
}
def record_interaction(self, student_input, dahlia_response):
"""Record this learning moment"""
session = {

"timestamp": datetime.now().isoformat(),
"student": student_input,
"dahlia": dahlia_response
}
self.memory["sessions"].append(session)
self.save_memory()
def save_memory(self):
"""Save to disk"""
with open(self.memory_file, 'w') as f:
json.dump(self.memory, f, indent=2)
def get_recent_context(self, n=5):
"""Get last N interactions for context"""
recent = self.memory["sessions"][-n:]
return "\n".join([f"Student: {s['student']}\nDahlia: {s['dahlia']}"
for s in recent])
```
### Step 5: Create Simple Launcher
**File: `launch_dahlia.py`**
```python
#!/usr/bin/env python3
import subprocess
import sys
from student_memory import StudentMemory
def chat_with_dahlia():
print("╔══════════════════════════════════════╗")
print("║ Welcome! I'm Dahlia 🌸 ║")
print("║ I'm here to help you learn. ║")
print("╚══════════════════════════════════════╝")
print()
# Get student name
name = input("What's your name? ").strip() or "student"
memory = StudentMemory(name)
print(f"\nHi {name}! What do you need help with today?")
print("(Type 'bye' when you're done)\n")

while True:
# Get student input
student_input = input(f"{name}: ").strip()
if student_input.lower() in ['bye', 'exit', 'quit']:
print("\nDahlia: Keep learning! I'll be here when you need me. 🌸")
break
if not student_input:
continue
# Get context from memory
context = memory.get_recent_context(3)
# Build prompt with context
prompt = f"""Previous conversation:
{context}
Student: {student_input}
Dahlia (respond as educational companion):"""
# Call Ollama
try:
result = subprocess.run(
['ollama', 'run', 'dahlia-edu', prompt],
capture_output=True,
text=True,
timeout=30
)
response = result.stdout.strip()
print(f"\nDahlia: {response}\n")
# Record this interaction
memory.record_interaction(student_input, response)
except subprocess.TimeoutExpired:
print("\nDahlia: Sorry, I'm thinking too slowly. Can you ask again?\n")
except Exception as e:

print(f"\nError: {e}\n")
if __name__ == "__main__":
chat_with_dahlia()
```

### Step 7: Format USB (if needed)
- Plug in USB
- Format as exFAT or FAT32 (works on all systems)

### Step 8: Copy to USB
**Find USB mount point:**
```bash
# In WSL, USB usually at:
ls /mnt/
# Look for your USB (like /mnt/d/ or /mnt/e/)
```

**Copy everything:**
```bash
USB_PATH="/mnt/e" # Adjust to your USB location
# Create structure
mkdir -p $USB_PATH/dahlia-edu
# Copy files
cp educational_dahlia.modelfile $USB_PATH/dahlia-edu/
cp student_memory.py $USB_PATH/dahlia-edu/

# Export the model
cd $USB_PATH/dahlia-edu/
ollama cp dahlia-edu dahlia-edu-usb
```

## Troubleshooting Keywords
**If Ollama not found:**
- Check Ollama installed: `ollama --version`
- Install if needed: [ollama.ai](https://ollama.ai)
**If model too slow:**
- Try smaller model: `smollm:360m` (229MB)
- Adjust in modelfile: `FROM smollm:360m`
**If USB not mounting:**
- Windows: Check Disk Management
- WSL: Try `sudo mount -t drvfs E: /mnt/e` (adjust letter)

**If memory not persisting:**
- Check write permissions: `ls -la memory_*.json`
- Verify USB not read-only
